Aljabar – Matematika SMP

Variabel? x? y? itu apa ya dalam matematika? Yuk kita bahas..

Materi

Aljabar adalah salah satu cabang utama matematika yang mempelajari struktur, relasi, dan operasi menggunakan simbol (biasanya huruf seperti \(x,y,a,b\)) untuk mewakili bilangan yang belum diketahui atau berubah-ubah (variabel).

Misalkan : 4x + 3y = 11

Contoh Soal

Soal: Jika x + 2y = 9 dan 2x + y = 12, maka nilai x + y adalah...

Untuk menjawab soal di atas, maka kita menggunakan sistem eliminasi dua persamaan.
x + 2y = 9
2x + y = 12
2x + 4y = 18
2x + y = 12
------------- -
3y = 6
y = 6/3
y = 2
x + 2y = 9
x + 2(2) = 9
x + 4 = 9
x = 9 - 4
x = 5
Maka nilai dari x + y adalah
= x + y
= 5 + 2
x + y = 7

Soal Latihan

  1. Jika ( x + y = 8 ) dan ( 2x + y = 11 ), maka nilai ( x + y ) adalah ...
  2. Jika ( 2x + y = 7 ) dan ( x + 3y = 11 ), maka nilai ( 2x + 2y ) adalah ...
  3. Jika ( x - y = 1 ) dan ( x + y = 9 ), maka nilai ( x \times y ) adalah ...
