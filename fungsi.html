Materi

Fungsi adalah hubungan antara dua variabel, di mana setiap nilai pada variabel pertama (x) memiliki tepat satu pasangan nilai pada variabel kedua (y).

x = variabel bebas
y = variabel terikat
f(x) = nilai fungsi untuk x tertentu
Masukan nilai x -- Diproses -- keluar nilai y


Bentuk Umum

f(x) = ax + b
Keterangan :
a = gradien
b = konstanta

contoh :
f(x) = 2x + 3

Contoh Soal

Soal: Diketahui f(x) = 2x + 5, tentukan nilah f(3)..

Langkah 1 :
Substitusi nilai x = 3 ke dalam fungsi.
f(3) = 2(3) + 5
f(3) = 6 + 5
f(3) = 11

Soal Latihan

  1. Jika f(x) = 3x - 4, tentukan nilai f(5).
  2. Jika f(x) = x2 + 2, tentukan nilai f(4).
  3. Diketahui f(x) = 4x + 1. Jika f(x) = 21, tentukan nilai x.
